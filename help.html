Eduscope Web Capture is a FHD screen recorder with WebCam and System Sound. It can output the recording as an MP4 downloadable locally. the special feature is the Embedded WebCam which can be placed at a desired location on screen.

1. Desktop - (Full screen, Application, Chrome Tab)

2. WebCam Only

Frame Rate: Frames Per Second

Camera Resolution: Width and Height in pixels(limited by camera capability)

Output Resolution: Screen will be captured at the maximum resolution and resized to the output resolution set here

Encoder: Encoder will decide the output file format and extension. (Example: H264 produces MP4

My recordings: Recordings from the current session are accessible from here. These are stored in memory. You may rename, view and delete the recordings from memory.

Help: This help document.

Recordings are limited to 1 hour.
After 1 hour, the recording will auto stop and preview will be displayed.
